对 Microsoft SQL Server 2005 JDBC Driver 执行的最基本的操作之一是建立到 SQL Server 数据库的连接。与数据库的所有交互都通过 SQLServerConnection 对象实现，并且由于 JDBC 驱动程序具有这种单一体系结构，因此几乎所有重要行为都涉及 SQLServerConnection 对象。

本部分中的主题说明如何建立到 SQL Server 数据库的连接以及如何使用该连接。
