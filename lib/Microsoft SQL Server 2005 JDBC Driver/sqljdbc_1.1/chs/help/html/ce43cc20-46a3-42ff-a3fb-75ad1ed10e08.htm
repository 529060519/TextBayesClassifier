使用 Microsoft SQL Server 2005 JDBC Driver 时，可用来建立到 SQL Server 数据库的连接的方法主要有两种。一种方法是在连接 URL 中设置连接属性，然后调用 DriverManager 类的 getConnection 方法来返回 SQLServerConnection 对象。

注意： 有关 JDBC 驱动程序所支持的连接属性的列表，请参阅设置连接属性。

第二种方法涉及到使用 SQLServerDataSource 类的 setter 方法设置连接属性，然后调用 getConnection 方法来返回 SQLServerConnection 对象。

此部分的主题说明了可用来连接到 SQL Server 数据库的各种方法，还说明了用来检索数据的各种技术。
