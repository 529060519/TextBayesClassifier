可用以下方式查找所安装的 Microsoft SQL Server 2005 JDBC 驱动程序的版本：

  • 调用 SQLServerDatabaseMetaData getDriverMajorVersion、getDriverMinorVersion 或 getDriverVersion 方法。

  • 版本出现在产品发布中的 readme.txt 文件中。

此外，可通过调用 SQLServerDatabaseMetaData 类的 getDriverName 方法返回 JDBC 驱动程序的名称。它将返回“Microsoft SQL Server 2005 JDBC 驱动程序”。

以下是调用 SQLServerDatabaseMetaData 类的各种方法的输出示例：

getDriverName = Microsoft SQL Server 2005 JDBC Driver

getDriverMajorVersion = 1

getDriverMinorVersion = 0

getDriverVersion = 1.0.xxx.x

其中“xxx.x”是最终版本号。
