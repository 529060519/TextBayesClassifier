Microsoft SQL Server 2005 JDBC Driver 示例应用程序是一组说明 JDBC 驱动程序的各种功能的应用程序。此外，这些示例应用程序还展示了在使用 JDBC 驱动程序和 SQL Server 数据库时可以参考的最佳编程实践。

所有示例应用程序都包含在 *.java 代码文件中，这些文件可以在本地计算机中编译和运行，它们位于以下位置的不同子文件夹中：

<安装目录>\sqljdbc_<版本>\<语言>\help\samples

此部分的主题说明了如何配置和运行示例应用程序，还包括了对示例应用程序所说明内容的讨论。

本节的相关信息

主题 说明

连接和检索数据

这些示例应用程序说明如何连接 SQL Server 数据库。它们还说明了可以从 SQL Server 数据库中检索数据的不同方法。

使用数据类型 (JDBC)

这些示例应用程序说明如何使用 JDBC 驱动程序数据类型方法来处理 SQL Server 数据库中的数据。

使用结果集

这些示例应用程序说明如何使用结果集来处理 SQL Server 数据库中包含的数据。
