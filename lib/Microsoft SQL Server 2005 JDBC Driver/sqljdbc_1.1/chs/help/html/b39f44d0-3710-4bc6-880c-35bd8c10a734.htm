Microsoft SQL Server 2005 JDBC Driver 的主要功能是允许 Java 开发人员访问 SQL Server 数据库中包含的数据。为了实现此功能，JDBC 驱动程序充当了 SQL Server 数据类型和 Java 语言类型以及对象之间的转换中介。

注意： 有关 SQL Server 和 JDBC 驱动程序数据类型（包括它们之间的差异以及如何将其转换到 Java 语言数据类型）的详细说明，请参阅了解 JDBC 驱动程序数据类型。

为了使用 SQL Server 数据类型，JDBC 驱动程序为 SQLServerPreparedStatement 和 SQLServerCallableStatement 类提供了 get<Type> 和 set<Type> 方法，为 SQLServerResultSet 类提供了 get<Type> 和 update<Type> 方法。要使用哪个方法取决于所使用的数据类型以及是否要使用结果集或查询。

此部分的主题说明了如何在 Java 应用程序中使用 JDBC 驱动程序数据类型来访问 SQL Server 数据。
