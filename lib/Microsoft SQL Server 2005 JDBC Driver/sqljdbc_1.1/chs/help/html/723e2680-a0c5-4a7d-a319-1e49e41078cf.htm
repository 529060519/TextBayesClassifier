无论应用程序的设计和开发如何成功，都将不可避免出现问题。一旦出现问题，具备一些诊断问题的技术很重要。使用 Microsoft SQL Server 2005 JDBC 驱动程序时，一些可能出现的常见问题是没有正确的驱动程序版本或无法连接到数据库。

此部分中的主题将介绍诊断这些问题和其他问题（包括错误处理、检查驱动程序版本、跟踪和解决连接问题）的大量技术。
