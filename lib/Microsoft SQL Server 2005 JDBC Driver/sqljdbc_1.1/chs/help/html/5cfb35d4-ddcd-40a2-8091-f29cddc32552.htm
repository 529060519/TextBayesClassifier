可以通过多种方式使用 Microsoft SQL Server 2005 JDBC Driver 来处理 SQL Server 数据库中的元数据。JDBC 驱动程序可用于获取数据库、结果集或参数的元数据。

JDBC 驱动程序提供了三个从 SQL Server 数据库中检索元数据的类：

  • SQLServerDatabaseMetaData，用于返回有关当前已连接的数据库的信息。

  • SQLServerResultSetMetaData，用于返回有关结果集的信息。

  • SQLServerParameterMetaData，用于返回有关预定义语句参数和可调用语句参数的信息。

本部分中的主题说明了如何使用这三个元数据类中的各个类来处理 SQL Server 数据库中的元数据。

注意： 就应用程序性能而言，本部分所讨论的元数据方法通常需要消耗大量资源，因此使用时应谨慎。
