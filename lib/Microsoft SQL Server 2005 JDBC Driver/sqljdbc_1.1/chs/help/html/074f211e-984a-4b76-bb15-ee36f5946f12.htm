此 1.1 版的 Microsoft SQL Server 2005 JDBC Driver 包含多个新功能和增强功能。以下各部分将说明此版本的 JDBC 驱动程序中的增强功能。

集成身份验证

JDBC 驱动程序目前支持在 Microsoft Windows 操作系统中使用“类型 2”集成身份验证。有关如何配置计算机以使用集成身份验证的信息，请参阅创建连接 URL。有关如何在 Java 应用程序中使用 integratedSecurity 连接字符串属性的信息，请参阅设置连接属性。

数据库镜像

JDBC 驱动程序目前支持使用 SQL Server 2005 数据库镜像，该功能可增加数据库可用性和数据冗余。有关如何通过 JDBC 驱动程序使用数据库镜像的详细信息，请参阅使用数据库镜像 (JDBC)。

包大小属性

JDBC 驱动程序目前支持使用 packetSize 连接字符串属性，使用该属性可以设置或检索用来与 SQL Server 进行通信的网络包大小。有关新 packetSize 属性的详细信息，请参阅设置连接属性。

示例应用程序

JDBC 驱动程序的帮助系统目前包含一个示例应用程序集，演示了在 Java 应用程序中使用 JDBC 驱动程序的各种技术。有关示例应用程序的详细信息，请参阅示例 JDBC 驱动程序应用程序。
