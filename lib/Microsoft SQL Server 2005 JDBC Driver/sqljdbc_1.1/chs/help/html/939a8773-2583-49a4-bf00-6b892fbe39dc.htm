Microsoft SQL Server 2005 JDBC Driver 是与 Java 数据库连接 (JDBC) 3.0 兼容的驱动程序，它可提供对 Microsoft SQL Server 2000 和 SQL Server 2005 数据库的可靠访问。JDBC 驱动程序可访问 SQL Server 2005 的许多新增功能，包括数据镜像、xml 数据类型、用户定义的数据类型以及大值数据类型；并且支持新的快照隔离游标类型。此外，JDBC 驱动程序还支持通过 SQL Server 2000 和 SQL Server 2005 使用集成身份验证。

此部分中的主题提供了对 JDBC 驱动程序的全面概述，包括使用的系统要求、如何使用，以及在何处可获取详细信息。
