安装指南
Microsoft SQL Server 2005 JDBC Driver 1.1
2006 年 8 月


简介
------------
以下是解压缩 sqljdbc_<版本>_enu.exe 和 sqljdbc_<版本>_enu.tar.gz 的指南。


WINDOWS
-------
1. 阅读 license.txt 文件。下载并使用 Microsoft SQL Server 2005 JDBC Driver 则表示接受此许可。

2. 将 sqljdbc_<版本>_enu.exe 下载到临时目录中。

3. 运行 sqljdbc_<版本>_enu.exe

4. 出现提示后，请输入安装目录；建议将此 zip 文件解压缩到 %ProgramFiles% 中的以下默认目录中：

	Microsoft SQL Server 2005 JDBC Driver

5. 将程序包解压缩之后，请打开 %安装目录%\Microsoft SQL Server 2005 JDBC Driver\sqljdbc_<版本>\<语言>\help\default.htm，
从而打开 JDBC 帮助系统。
Web 浏览器中将显示该帮助系统。


UNIX
----
1. 阅读 license.txt 文件。下载并使用 Microsoft SQL Server 2005 JDBC Driver 则表示接受此许可

2. 将 sqljdbc_<版本>_enu.tar.gz 下载到临时目录中。

3. 若要将通过 zip 压缩的 tar 解压缩，请导航至要存放驱动程序解压缩文件的目录，然后键入：

	gzip -d sqljdbc_<版本>_enu.tar.gz

4. 若要将 tar 解压缩，请将其移动到驱动程序安装的目标目录，然后键入：

	tar -xf sqljdbc_<版本>_enu.tar

5. 将程序包解压缩之后，请打开 %安装目录%/Microsoft SQL Server 2005 JDBC Driver/sqljdbc_<版本>/<语言>/help/default.htm，
从而打开 JDBC 帮助系统。
默认 Web 浏览器中将显示该帮助系统。
